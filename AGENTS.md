# Agent Instructions: Pascal Heus Personal Website (ph-www)

This repository contains the source code and design specifications for the personal professional website of **Pascal Heus**. As an agent working in this repo, you are responsible for maintaining a state-of-the-art, high-performance, and visually stunning web presence.

## 🚀 Vision & Mission
Create a "WOW" factor one-page responsive application that highlights Pascal's 40 years of IT expertise, specializing in data, metadata (DDI/FAIR), and AI. The site should feel like a premium digital experience, blending technical depth with creative flair.

## 🛠 Technical Stack
- **Framework**: [11ty (Eleventy)](https://www.11ty.dev/) for static site generation.
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) for modern, responsive layouts.
- **Interactivity**: [Alpine.js](https://alpinejs.dev/) for lightweight client-side logic.
- **Visuals/3D**: [Three.js](https://threejs.org/) & [Drei](https://github.com/pmndrs/drei) for immersive 3D animations and data visualizations.
- **Deployment**: GitHub Pages via GitHub Actions.

## 🎨 Design Principles
- **Aesthetics**: Premium, modern, and high-tech. Use a mix of glassmorphism, subtle gradients, and dark/light modes.
- **3D Interaction**: Use Three.js to represent data connections, knowledge graphs, or abstract digital structures.
- **Responsiveness**: Mobile-first design; must look flawless on all devices.
- **Theming**: Implement a robust theme switcher (Light/Dark/System) with a sleek UI.
